撰寫新的遠征

發布《無盡的遠征》到 GitHub Pages

這是一個完全免費、永久有效、無任何廣告的網站託管方式。只需要跟著以下 4 個簡單步驟，就能擁有您專屬的永久文字網站！

Step 1

複製當前網頁原始碼

點擊右側程式碼編輯器，選取並複製整段 HTML 原始碼。我們所有的自訂文章皆保存在瀏覽器緩存中，部署時也可以先將喜愛的文章直接填入預設代碼裡。

Step 2

註冊並登入 GitHub

前往 github.com 免費註冊一個帳號。

Step 3

創建新儲存庫 (Repository)

在 GitHub 首頁右上角點選 + New，並將儲存庫**完全命名為**： 你的使用者名稱.github.io

Step 4

建立並發布 index.html

在新建的儲存庫點選 "creating a new file"，將新檔案命名為 index.html，將剛才複製的所有網頁原始碼貼進去，並點擊下方 "Commit changes"。

遠征完成！

等待約 1~2 分鐘，即可在瀏覽器輸入您的專屬網址： https://你的使用者名稱.github.io，就能立刻看見屬於您的「無盡的遠征」文字手札！
